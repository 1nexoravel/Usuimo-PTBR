## 🌐 Plataforma de Tradução
As traduções estão sendo gerenciadas via Weblate:
https://weblate.inexo.cloud/projects/a-simple-life-with-my-unobtrusive-sister/#components

> ⚠️ Devido à limitação de 60 mil palavras no Crowdin, os arquivos e traduções foram migrados para o Weblate.

Para contribuir pelo Weblate, é necessário realizar um cadastro com e-mail.
